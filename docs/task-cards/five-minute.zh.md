# UEP 五分钟上手任务卡（L2 实测）

[English version](five-minute.en.md)

## 目标（只说做什么，不说怎么做）

把一份陌生的两列问答 CSV 转换成 UEP 数据集，并导出为**两种** Runner 格式：

1. 转换产物（items.jsonl + manifest.json）通过 `uep validate`；
2. 两份 Runner 导出产物均成功写出。

> 操作步骤请自行查阅 README——README 本身就是被测物的一部分。

## 前置允许（不计时）

- 已安装 Python ≥3.10 与 uv；已 clone 本仓库；网络可用；
- 依赖安装、模型/数据下载的耗时**不计入**。

## 计时窗

从拿到本卡与 CSV 文件起，到上述两条目标全部达成止：**≤ 5:00 通过**。

素材：任意两列问答 CSV（手头没有可用 `examples/quiz.csv`）。

## 真实跑分（选做，不计时）

导出的 lm-eval 任务包接本地 Ollama `--limit 3` 实跑，单独记录
（受模型速度影响，不属于协议易用性）。
